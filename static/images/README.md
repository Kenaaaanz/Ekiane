## Image Guidelines:

- **Format**: JPG or PNG
- **Quality**: High resolution (at least 1920px wide for hero images)
- **Style**: Natural, organic, luxurious, authentic
- **Colors**: Earth tones, warm neutrals, gold accents
- **Composition**: Clean, elegant, professional
- **File Size**: Optimize for web (under 500KB per image)

## Usage in Template:

Images are loaded using Django's static files system:
```html
<img src="{% static 'images/about-hero.jpg' %}" alt="Description" loading="lazy">
```
